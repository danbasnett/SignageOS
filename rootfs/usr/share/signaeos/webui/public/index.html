NDI Devices

Add the IP address of any NDI device — camera, encoder, ATEM etc. The NDI SDK will query it directly, returning all its streams regardless of subnet.

NDI Discovery Server optional

If you run an NDI Discovery Server on your network, set its IP here. All registered sources will be discovered automatically.

Manual Sources add by NDI name

If you know the exact NDI source name (as shown in NDI Video Monitor), add it here directly. Format is usually Device Name (IP) e.g. ATEM MINI (10.32.40.2)
