Companion Server

Setup Guide

  1. Open Bitfocus Companion on your PC
  2. Go to Connections → Add → search Generic HTTP
  3. For each URL, create a button with action:
    POST http://<this-device-ip>:3000/api/switch/0 (0, 1, 2… per URL)
  4. Or use Satellite connection type — Companion Satellite is running on port 16622 on this device
  5. Plug Stream Deck into this box or into your PC running Companion
